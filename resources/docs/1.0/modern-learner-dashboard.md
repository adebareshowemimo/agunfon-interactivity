# Modern Learner Dashboard

---

**Modern Learner Dashboard** (`block_modernlearnerdashboard`) gives every Moodle learner a single, personalized home. In one block on the Dashboard they see their courses, completion, due-soon activities, grades, badges and certificates — plus a learning transcript, a searchable course library, and inline profile editing. It runs entirely on core Moodle data, with no external service and nothing for the learner to set up.
