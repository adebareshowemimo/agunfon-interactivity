<a name="what"></a>
## What it is

The **Course Library** tab turns the dashboard into a discovery surface: it shows visible courses the learner is **not yet enrolled in**, so the next course to take is one click away. Switch the filters to review all courses instead of only un-enrolled ones.

<a name="filters"></a>
## Search, filter and sort

The library has a full course browser:

- **Search** by title, short name or category.
- **Filter** by category and by enrolment status.
- **Sort** by title or category.
- **Paginate** large catalogues, with a page-size that follows the dashboard.

The **My courses** tab reuses the same browser for enrolments, adding status (in progress / not started / completed) and sort-by-progress or recent-activity. A **grid / list** toggle switches the layout in both.

<a name="stats"></a>
## Catalogue stats

The library header surfaces catalogue counts — available courses, categories, activities and (when Course Certificate is installed) certificates — so learners get a sense of what's on offer.

<a name="enrol"></a>
## Self-enrolment

Where a course allows it, learners can **enrol straight from the library** and start immediately — turning browsing into enrolment without administrator involvement, and driving uptake of optional and compliance courses.
