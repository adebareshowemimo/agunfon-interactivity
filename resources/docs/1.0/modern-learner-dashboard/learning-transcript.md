<a name="what"></a>
## What it is

The **Learning transcript** is a completed-training record for the learner — a single panel that lists every course they have finished, with the evidence an administrator or compliance reviewer needs. It turns the dashboard into an at-a-glance audit view, without leaving the learner's home.

<a name="rows"></a>
## What each row shows

Each completed course appears as a row with:

| Column | Meaning |
|---|---|
| Course | The completed course name (links to the course) |
| Category | The course's category |
| Completion date | When the course was completed |
| Activity completion | Completed vs total tracked activities (e.g. *8 of 10*) |
| Grade average | The learner's grade for the course |
| Certificate | A link to the issued Course Certificate *(when available)* |
| Open | Jump straight into the course |

<a name="where"></a>
## Where to find it

Open the **My courses** tab and use the **Learning transcript** button to expand the panel. It sits alongside the My courses stats and course browser — see [Dashboard Tabs](/{{route}}/{{version}}/modern-learner-dashboard/dashboard-tabs#mycourses).

<a name="certs"></a>
## Certificates in the transcript

The **Certificate** column is populated from the **Course Certificate** activity (`mod_coursecertificate`). If that plugin isn't installed, the column is omitted and the rest of the transcript works unchanged. See [Requirements](/{{route}}/{{version}}/modern-learner-dashboard/requirements#certificates).
