<a name="skins"></a>
## Available skins

Premium ships seven player skins:

| Skin | Look |
|---|---|
| **Classic** | The default Modern Video Player look. |
| **YouTube** | YouTube-inspired controls and layout. |
| **Vimeo** | Vimeo-inspired clean styling. |
| **Netflix** | Cinematic, dark, big-play styling. |
| **Plyr** | Plyr-style minimalist controls. |
| **Apple** | Apple-inspired glassy controls. |
| **Minimal** | Stripped-back, distraction-free. |

> {primary} The **tracking and enforcement engine is identical across all skins** — only the appearance changes. Switching skins never affects completion, grading, or integrity.

<a name="default"></a>
## Setting a site default

An administrator sets the default skin for new activities:

**Site administration → Plugins → Activity modules → Modern video player → Appearance**

- **Enable player styles** — turns the skin picker on (on by default).
- **Default player style** — the skin applied to new activities (default: *classic*).

<a name="per-activity"></a>
## Choosing a skin per activity

When player styles are enabled, teachers can override the skin on each activity in the activity form. If the picker is disabled site-wide, all activities use the site default.
