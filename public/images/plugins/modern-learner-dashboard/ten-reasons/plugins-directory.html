Modern Learner Dashboard

Modern Learner Dashboard gives every learner one focused home on the Moodle Dashboard. Instead of hunting across My courses, the gradebook, the calendar and their profile, learners see enrolled courses, completion, due-soon activities, grades, badges and certificates in a single, tabbed block.

It runs entirely on core Moodle APIs — no external service, no new database tables, no cron tasks — surfaces a searchable course library for self-enrolment, lets learners edit their own profile inline, and is theme-aware, fast (lazy-loaded tabs) and privacy-clean.

Supports Moodle 4.5–5.2.

Moodle 4.5–5.2 GPLv3 Stable
